# Project Knowledge Base

Эта папка хранит проектную документацию, которая описывает архитектуру всего PitGO, идеи реализации и зависимости.

## Содержимое

- [service-map.md](service-map.md) - компактная схема сервисов PitGO: MVP и future.
- [dependencies.md](dependencies.md) - зависимости backend, crates и frontend.

## Правила ведения

- Фиксировать только решения и идеи, которые важны для развития проекта.
- Не дублировать доменную документацию из [../domain](../domain), а ссылаться на нее.
- Обновлять документы вместе с изменениями архитектуры, зависимостей или ключевых технических решений.
- Держать записи короткими: подробности лучше переносить в отдельные файлы внутри этой папки.
